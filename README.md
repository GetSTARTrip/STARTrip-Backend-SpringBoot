# Startrip Backend
빅데이터 기반의 명소 큐레이팅 서비스

## Installing
다음 명령을 실행해 프로젝트 패키지를 설치하고 로컬 서버를 실행합니다.
```shell
$ ./gradlew bootRun
```

## Build
`./gradlew build`를 실행하여 프로젝트를 빌드합니다. 빌드 결과물은 build/libs/ 디렉터리에 저장됩니다.

## 사용기술
Backend `Java11` `Spring Boot 2.6.2`

DB `PostgreSQL`

CI `GitAction`

Deploy `Docker`
